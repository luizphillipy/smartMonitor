# smartMonitor
This project is intended to control and monitor smart devices at Home
- The system should discover devices connected to the home network
- The system should allow the user to search for a device on the network by IP, MAC address, and device name
- Every device should have a name, IP, MAC address, model, serial, tw version, brand, latency, status, and communication ports (TCP/IP)
- Every device should have a skill that is divided into basic: turn on, turn off, restart, and specialized that depends on the device's capability
- The system should check the status every 1 min, and it is user-defined
- The system should allow the user to add a new device manually
- The system should allow the user to browse devices
- Devices compatible with version 1: TP-link smart plugs, Spectrum modem, Spectrum Wi-Fi, Toshiba smartTV, smart fan from xxxxx
